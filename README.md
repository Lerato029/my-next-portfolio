## Getting Started

First, install node modules:

```bash
npm install

```
Then, run the development server:

```bash
npm run dev

```
App can now be viewed on: [http://localhost:3000](http://localhost:3000) 

Push Work To Github:
git add --all

git commit -m "First commit"

git push  origin
